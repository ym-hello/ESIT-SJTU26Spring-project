# S800 PC Host

S800 时钟上位机，基于 PyQt5 实现。通过串口与 S800 下位机通信，提供数字孪生面板、参数控制、事件日志和数据可视化等功能。

## 功能概览

- **串口通信** — 自动扫描可用串口，支持多种波特率，收发日志实时显示
- **数字孪生** — 8 位七段数码管、8 颗 LED 指示灯、按键面板，实时同步下位机状态
- **时钟控制** — 读取/设置日期、时间、闹钟，切换日间/夜间显示模式
- **天气推送** — 通过天气 API 获取数据并下发到下位机
- **语音播报** — 基于 edge-tts / pyttsx3 的事件语音播报
- **事件日志** — 结构化 CSV 日志记录，支持导出
- **数据面板** — matplotlib 图表展示闹钟命中率等统计数据
- **自动日夜** — 根据经纬度日出日落时间自动切换显示模式

## 环境要求

- **Python 3.11+**（推荐 3.11）
- **Windows**（依赖 pywin32、comtypes 等 Windows 专用包）
- 串口驱动（CH340 / CP2102 等，视硬件而定）

## 安装

```bash


# 1. 创建虚拟环境
python -m venv .venv

# 2. 激活虚拟环境
.venv\Scripts\activate       # Windows CMD/PowerShell
# source .venv/Scripts/activate  # Git Bash

# 3. 安装依赖
pip install -r requirements.txt
```

## 运行

```bash
# 确保虚拟环境已激活
python main.py
```

启动后会打开主窗口，包含以下区域：

| 区域 | 说明 |
|------|------|
| 左上 - 数字孪生 | 七段数码管显示、LED 状态、虚拟按键 |
| 左下 - 控制面板 | 串口连接、日期/时间/闹钟设置、天气推送 |
| 右侧 - 日志面板 | 串口收发记录，支持筛选和导出 |
| 右侧 - 数据面板 | 统计图表（标签页切换） |

### 使用流程

1. 在控制面板选择串口号和波特率，点击 **连接**
2. 连接成功后，上位机会自动查询下位机当前状态
3. 通过控制面板修改时间、闹钟、显示模式等参数
4. 日志面板实时显示所有收发消息
